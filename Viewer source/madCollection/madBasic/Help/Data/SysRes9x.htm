System Resources (9x)

The function "GetFreeSystemResources" returns the amount of free system resources in percent. You can ask the system, GDI or user resources. These are limited in win9x/ME, but not that way in winNT/2k/XP. So this function makes only sense in win9x/ME.

// system : resource = 0
// GDI    : resource = 1
// user   : resource = 2
function GetFreeSystemResources (resource: word) : word;

// Example:
ShowMessage('free system resource: ' + GetFreeSystemResources(0) + '%.');
